# Amaze Tech Solutions — TMS Back-Office Services Website

Marketing website for [tms.amazetech.net](https://tms.amazetech.net) — expert TMS back-office services for trucking companies in Canada and USA.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS
- **Language:** TypeScript
- **Deployment:** Static export / Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```
